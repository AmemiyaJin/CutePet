# CutPet - 桌面宠物应用 🐹

一个基于 C++ 和 GDI+ 开发的 Windows 桌面宠物应用。

## ✨ 功能特性

- 🐹 **桌面宠物**：透明置顶窗口，仓鼠在桌面上自由活动
- 🖱️ **拖拽移动**：鼠标拖拽移动宠物位置
- 🥕 **喂食系统**：点击喂食，仓鼠吃胡萝卜
- 😴 **自动睡觉**：闲置一段时间后自动进入睡眠状态（Zzz 动画）
- ❤️ **抚摸互动**：左键点击抚摸，附带爱心粒子特效
- 🎾 **玩耍系统**：消耗玩具道具进行玩耍
- 📊 **等级系统**：挂机和经验积累，5个进化阶段
- 🎨 **视觉效果**：双缓冲渲染、粒子特效、呼吸动画、眨眼效果

## 🛠️ 技术栈

- **语言**：C++17
- **图形**：GDI+
- **窗口**：Win32 API
- **构建**：Visual Studio 2022

## 🚀 编译运行

1. 用 Visual Studio 2022 打开 `CutPet.sln`
2. 选择 `x64` + `Release` 配置
3. 按 `Ctrl+Shift+B` 编译
4. 按 `F5` 运行
